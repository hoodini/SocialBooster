## 🛠️ התקנה

### שלב 1: הורדת התוסף
```bash
git clone https://github.com/yuvalav/yuv-ai-socialbot-pro.git
cd yuv-ai-socialbot-pro
```

### שלב 2: התקנה בכרום
1. פתח את Chrome ועבור ל: `chrome://extensions/`
2. הפעל את "Developer mode" (מצב מפתח)
3. לחץ על "Load unpacked" (טען בלתי ארוז)
4. בחר את התיקייה שהורדת

### שלב 3: קבלת API Key של Cohere
1. עבור ל: [Cohere Dashboard](https://dashboard.cohere.com/)
2. צור חשבון או התחבר
3. עבור ל-API Keys
4. צור API Key חדש
5. העתק את המפתח

---

## ⚙️ הגדרה ראשונית

### 1. הגדרת API Key
1. לחץ על איקון התוסף בסרגל הכלים
2. הדבק את ה-API Key של Cohere
3. לחץ "שמור" - התוסף יבדוק את החיבור

### 2. יצירת פרסונה ראשונה
1. לחץ על "הוסף" בסעיף הפרסונות
2. הזן שם לפרסונה (למשל: "מקצועי ידידותי")
3. כתב תיאור של האופי והטונציה
4. הוסף דוגמאות לכתיבה (לפחות 2-3)
5. שמור את הפרסונה

### 3. הפעלת האוטומציה
1. בחר את הפרסונה הרצויה
2. סמן "לייקים אוטומטיים" ו/או "תגובות אוטומטיות"
3. ודא שהפלטפורמה מופעלת (LinkedIn/Facebook)

## 🎯 איך זה עובד?

### זיהוי פוסטים והתנהגות
- התוסף מזהה אוטומטי פוסטים חדשים
- מבחין בין גלילה רגילה לקריאה מעמיקה
- **1.5 שניות צפייה** = לייק אוטומטי
- **3 שניות צפייה** = תגובה אוטומטית

### יצירת תגובות
1. התוסף שולח את תוכן הפוסט ל-Cohere API
2. המודל מייצר תגובה בהתאם לפרסונה הנבחרת
3. התגובה מוזרקת לתיבת הטקסט
4. **המשתמש יכול לערוך ולאשר לפני פרסום**

### תגובות על תגובות
- לחיצה על "Reply" מפעילה אוטומטית את המערכת
- התוסף שולח גם את הפוסט המקורי וגם את התגובה
- מייצר תגובה רלוונטית להקשר המלא
